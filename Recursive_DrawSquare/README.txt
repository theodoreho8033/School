RECURSIVE DRAW SQUARE 

  Description: Given specs draw squares in pattern 
  
  Square.c: Square application
  
  Input: stdin string specifying pattern: 

    Input Format: 
      {dimension x of image} {dimension y of image} {recursive depth} {offset of next square} {start x} {start y} {end x} {end y}
      
   Output: .png file of created image
